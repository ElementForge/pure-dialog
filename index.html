pure-dialog demo

This won't show anything visually. You need to use JavaScript APIs to show / hide it. 

<pure-dialog id="example" title="Pure Dialog Demo" buttons="Yes, No">Will you star this repo?</pure-dialog>
        
<button onclick="document.querySelector('#example').showModal()">show</button>
<button onclick="document.querySelector('#example').showModal()">showModal</button>
Will you star this repo?
